# UniPipelines

## Why you should use it
- Declarative strict config
- Scaffolding
- Fully typed
- Python support 3.6+ 
- Brokers support
    - kafka
    - rabbitmq
    - inmemory simple pubsub
- Interruption handling = safe user code transactions
- CLI
